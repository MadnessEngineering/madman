# Contributing

Improvements to the SKILL.md prompt are welcome — open a PR with before/after examples showing the change.

## How

1. Fork repo
2. Edit `skills/mad-tinker/SKILL.md` — this is the only copy you need to touch
3. Open PR with:
   - **Before:** what tinker say now
   - **After:** what tinker say with change
   - One sentence why change better

> **Note:** `mad-tinker/SKILL.md`, `plugins/mad-tinker/skills/mad-tinker/SKILL.md`, `.cursor/skills/mad-tinker/SKILL.md`, and `mad-tinker.skill` are auto-synced by CI after merge. Do not edit them directly.
> 
> **Note on compress skill:** If you are modifying the compress skill, edit `tinker-compress/SKILL.md` or `tinker-compress/scripts/`. CI will automatically sync these changes to `skills/compress/` and `plugins/mad-tinker/skills/compress/`.

Small focused change > big rewrite. Mad Tinker like simple.
